### 👁️ Currently building: [Sacred Geometry Visualizer](link-to-demo-here)
A Stella Octangula rotates inside a mandala, and both morph based on the harmonic content of whatever audio you feed it — pitch, intervals, amplitude, overtones. Upload a file and the whole thing gets pre-analyzed so visuals land ahead of the sound.

*React, raw WebGL2, GLSL, Web Audio API, pitchy for pitch detection.*

[Live demo](https://sacred-geometry-ivory.vercel.app/) · [Source](https://github.com/mostafammagdy/sacred-geometry)
